# Ellis Thomas Senior Project Abstract
The Southeastern Pennsylvania Transportation Authority (SEPTA) bus network service is inefficient, complicated, and infrequent. SEPTA is currently in the process of redesigning the bus lines to improve reliability and frequency, and expand the reach of the public transit network in Philadelphia. However, SEPTA has done minimal work to evaluate how well the current bus system provides access to childcare centers for working parents who depend on the bus for commuting. I developed the SEPTA-Childcare Access Dashboard in order to help users understand the level of access SEPTA provides to childcare locations throughout Philadelphia. Built using Node.js and Embedded JavaScript Templates, the dashboard provides insights for transit-dependent parents, owners of childcare businesses, SEPTA data analysts, and others. After development, the dashboard was evaluated through user testing and a survey.
